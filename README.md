# TripleZ's dotfiles

This repo is for my dotfiles.

Currently this repo is using [mackup](https://github.com/lra/mackup) to backup and restore dotfiles.

## Usage

### Backup

```bash
$ mackup backup [--dry-run]
```

### Restore

```bash
$ git clone git@github.com:Triple-Z/dotfiles.git
$ mackup restore
```
